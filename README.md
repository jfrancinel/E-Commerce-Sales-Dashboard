# E-Commerce Sales Dashboard

## Project Overview
This project analyzes e-commerce sales data using Microsoft Excel to identify sales trends, product performance, and regional sales distribution. An interactive dashboard was created to provide meaningful business insights and support data-driven decision-making.

## Objectives
- Analyze overall sales performance
- Identify top-performing products and categories
- Compare sales across different regions
- Monitor sales trends over time
- Build an interactive dashboard for data exploration

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Key Performance Indicators (KPIs)
- Total Sales Revenue
- Total Profit
- Total Quantity Sold

## Dashboard Features

### Sales Trend Analysis
- Monthly sales trend visualization
- Identification of peak and low-performing months

### Product Performance Analysis
- Top-selling products
- Product sales comparison by category

### Category Analysis
- Sales distribution across product categories
- Category performance comparison

### Interactive Filtering
- Category slicer
- Region slicer
- Date slicer
