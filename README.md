# E-Commerce-Sales-Dashboard

Project Overview

This project analyzes e-commerce sales data using Microsoft Excel. The goal is to identify sales trends, product performance, and regional sales distribution through an interactive dashboard.

Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Data Cleaning
Dashboard Design
KPIs
Total Sales Revenue
Total Profit
Total Orders
Total Quantity Sold
Dashboard Features
Monthly Sales Trend
Sales by Category
Sales by Region
Top Products Analysis
Interactive Slicers
